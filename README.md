## Instrucciones de instalación

1. Abrir el terminal y ejecutar los siguientes comandos
   1. cd PruebaTM
   2. npm install
   3. npx expo start
2. En el terminal deberia aparecer un QR para ejecutar la aplicacion con Expo Go
